REST API reference

Connect your mobile app to these endpoints. Base URL: http://localhost:3000/api/v1

POST /reports

Submit a garbage sighting with an image and GPS coordinates. The server maps coordinates to the nearest tracked city within a 25 km radius.

Content-Type: multipart/form-data

image        file     Required — JPEG/PNG/WebP ≤ 10 MB
latitude     float    Required (unless city_id provided)
longitude    float    Required (unless city_id provided)
city_id      string   Optional — overrides GPS lookup
severity     string   "low" | "medium" | "high"
notes        string   Optional description
// 201 Created

GET /cities

Returns the ranked list of all tracked cities with cleanliness scores.

?state=Maharashtra   filter by state name
?sort=score|reports  sort order (default: score)
?limit=50            max results (default: all)
GET /cities/:id

Returns a single city by its 3-letter ID (e.g. MUM, DEL).

GET /cities/:id/reports

Paginated recent reports for a city. Ideal for city drill-down screens in the mobile app.

?page=1       page number (default: 1)
?per_page=20  results per page (default: 20)
GET /heatmap

All individual report points as lat/lng + intensity. Use with Leaflet.heat or Mapbox GL heatmap layers on mobile.

GET /heatmap/cities

City-level aggregated heatmap data — one point per city with a 0–1 intensity value.
